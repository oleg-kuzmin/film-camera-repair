# Film Camera Repair

Сайт-лэндинг по ремонту фотоаппаратов.

## Функционал

- Ввод почты и отправка формы.

## Технологии

- Семантическая структура html.
- Адаптивная верстка Perfect Pixel по макету Figma от 320px.
- Структура файлов, наименование классов в стиле BEM.
- Препроцессор Sass.
- Используется Flex.
